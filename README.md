# 🎮 GamePitch AI

> **AI-powered marketing for indie game developers.**

GamePitch AI is an AI-powered platform that helps indie developers and small game studios create professional marketing content in seconds.

Many developers build amazing games but struggle to communicate their value. Marketing often requires skills, time, and budget that independent creators simply don't have.

GamePitch AI solves that problem by generating high-quality, game-specific marketing content using Large Language Models (LLMs), allowing developers to focus on what they do best: building great games.

---

## ✨ Features

Current MVP includes:

- 🎮 Steam Description Generator
- 📋 Copy-to-clipboard support
- 🔄 Content regeneration
- 🤖 AI-powered content generation
- ⚡ Fast and intuitive user experience

Future releases will include:

- Steam Tag Generator
- Social Media Generator
- Trailer Script Generator
- Publisher Pitch Generator
- Kickstarter Campaign Generator
- Email Outreach Generator
- Multi-language support
- AI-powered marketing recommendations

---

## 🚀 Why GamePitch AI?

Unlike general-purpose AI assistants, GamePitch AI is designed specifically for the game development industry.

It understands:

- Game genres
- Core gameplay mechanics
- Steam store conventions
- Marketing language
- Indie game positioning
- Publisher communication

The goal is not simply generating text.

The goal is generating marketing content developers can actually use.

## 🏗️ Architecture

GamePitch AI is being built as a production-ready SaaS application following modern software engineering practices.

### Technology Stack

| Layer      | Technology          |
| ---------- | ------------------- |
| Frontend   | React               |
| Backend    | Python + FastAPI    |
| Database   | PostgreSQL          |
| AI         | OpenAI / OpenRouter |
| Containers | Docker              |
| CI/CD      | GitHub Actions      |

### Engineering Principles

- Clean Architecture
- API-First Design
- Domain-Driven Design concepts
- Dependency Injection
- Structured Logging
- Observability
- Automated Testing
- Security by Design
- Feature Flags
- Scalability
